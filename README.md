# YouTube Clone - React & YouTube Data API

A fully functional, responsive YouTube clone built with React, CSS, and the YouTube Data API v3. 

## 🚀 Features

- **Real-Time Data**: Integrated with YouTube Data API v3 for live trending videos and search results.
- **Search & Playback**: Functional search bar and a custom video player with in-app navigation.
- **Theme Toggle**: Light and Dark mode implementation using binary XOR logic (`theme ^ 1`).
- **Responsive Design**: Two-column layout on desktop, stacking vertically on mobile devices.
- **Modular Components**: Clean architecture with reusable `Navbar`, `Sidebar`, `VideoGrid`, and `VideoCard`.
- **Recommended Sidebar**: In-app recommendations next to the player for seamless browsing.

## 🛠️ Tech Stack

- **Frontend**: React (Functional Components, Hooks)
- **Styling**: Vanilla CSS (Custom Design System)
- **API Handling**: Axios
- **Icons**: Lucide React

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/e25b070630-stack/YOUTUBECLONE.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up API Key**:
   Create a `.env` file in the root directory and add your YouTube API Key:
   ```env
   REACT_APP_YOUTUBE_API_KEY=your_api_key_here
   ```

4. **Run the app**:
   ```bash
   npm start
   ```
